This is a collection of pokedex text data scraped from pokemondb.net
fetch-dex.py isa web scraper that scrapes the aforementioned text data.

fetch-dex.py makes use of requests and beautiful soup to identify and isolate desired data.
This collection is intended for personal use and project "Pokedexto," a word guessing game based on the popular web game contexto using pokedex entries as the corpus.
